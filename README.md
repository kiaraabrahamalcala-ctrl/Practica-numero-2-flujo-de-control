# Practica numero 2 flujo de control
Materia: Lógica de Programación  
Profesor: Gamalier Reyes del Carmen  
Integrantes: Kiara Vásquez, Carlos David

Descripción
Programa en C++ que calcula el Impuesto Sobre la Renta (ISR) de un empleado
en República Dominicana, según la escala vigente de la DGII para el año 2026.
El programa solicita el sueldo mensual, descuenta la TSS (5.91%), anualiza
el monto y aplica la tabla progresiva de tramos (15%, 20%, 25%) para calcular
el descuento mensual de ISR. Si el sueldo anualizado no supera el mínimo
exento (RD$416,220), el programa muestra "0 (N/A)".
